A. Not Found

题目描述

给你一个长度在 111 和 252525 之间的字符串 SSS ，它由小写英文字母组成。
请输出一个没有出现在 SSS 中的小写英文字母。
如果有多个这样的字母，可以输出其中任意一个。

B.Grid Rotation

题目描述

有两个网格 SSS 和 TTT ，每个网格有 NNN 行和 NNN 列。让 (i,j)(i,j)(i,j) 表示从上往下数第 iii 行和从左往上数第 jjj 列的单元格。

网格 SSS 和 TTT 中的每个单元格都被涂成白色或黑色。如果 Si,jS_{i,j}Si,j​ 为"."，则 SSS 的单元格 (i,j)(i,j)(i,j) 为白色；如果 Si,jS_{i,j}Si,j​ 为 “#”，则单元格 (i,j)(i,j)(i,j) 为黑色。同样的情况也适用于 TTT 。

您可以按任意顺序执行以下两种类型的运算任意多次。求使网格 SSS 与网格 TTT 相同所需的最少运算次数。

  • 选择网格 SSS 中的一个单元格并改变其颜色。
  • 顺时针旋转整个网格 SSS 909090 度。

C.Cycle Graph?

问题陈述

给你一个简单的无向图，图中有 NNN 个顶点和 MMM 条边。顶点编号为 1,2,…,N1,2,\dots,N1,2,…,N ，边编号为 1,2,…,M1,2,\dots,M1,2,…,M 。边 iii 连接顶点 AiA_iAi​ 和 BiB_iBi​ 。

请判断该图是否为循环图。

简单无向图的定义： 简单无向图是指不存在自循环或多条边的无向边图。

循环图的定义： 如果存在 (1,2,…,N)(1,2,\dots,N)(1,2,…,N) 的排列 (v1,v2,…,vN)(v_1,v_2,\dots,v_N)(v1​,v2​,…,vN​) ，且其顶点标记为 1,2,…,N1,2,\dots,N1,2,…,N ，则一个 NNN /顶点图是循环图：

  • 对于每个 i=1,2,…,N−1i = 1,2,\dots,N-1i=1,2,…,N−1 ，顶点 viv_ivi​ 和 vi+1v_{i+1}vi+1​ 之间有一条边。
  • 顶点 vNv_NvN​ 与 v1v_1v1​ 之间有一条边。
  • 不存在其他边。
