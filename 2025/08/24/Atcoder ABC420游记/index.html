B.Most Minority

题目描述

1,2,…,N1,2,\dots,N1,2,…,N 人（其中 NNN 为奇数）进行了 MMM 次投票，每个人都选择了 "0 "或 “1”。
每个人在每一轮投票中的票数是由长度为 MMM 的 NNN 字符串 S1,S2,…,SNS_1,S_2,\dots,S_NS1​,S2​,…,SN​ 给出的，字符串由 "0 "和 "1 "组成，其中 SiS_iSi​ 的 jjj -th字符代表 iii 人在 jjj -th投票中的投票内容。

在每次投票中，处于少数的人获得 111 个点。
更确切地说，点数是根据以下规则给出的：

  • 假设 xxx 人在投票中选择了 “0”， yyy 人选择了 “1”。
    • 如果有 x=0x=0x=0 或 y=0y=0y=0 人选择 “1”，则每个人都会因这一投票得到 111 分。
    • 否则，如果是 x<y ，只有在该投票中投了 "0 "票的人才会得到 111 分。
    • 否则，只有投 "1 "票的人才能得到 111 分。
    • 请注意，由于 NNN 是奇数，所以 x=yx=yx=y 永远不会出现。

在完成 MMM 投票后，找出所有在这些投票中总得分最高的人。

思路

直接模拟即可。

D.Toggle Maze

题目描述

有一个网格，网格中有 HHH 行和 WWW 列。让 (i,j)(i,j)(i,j) 表示位于从上往下第 iii 行和从左往上第 jjj 列的单元格。每个单元格的状态用字符 Ai,jA_{i,j}Ai,j​ 表示，其含义如下：

  • . ：空单元格。
  • # : 障碍单元格。
  • S ：起始单元格。
  • G : 目标单元格。
  • o : 打开的单元格。
  • x : 关闭的单元格。
  • ? : 开关单元。

高桥可以通过一次操作从当前单元格移动到既不是障碍物单元格也不是封闭门单元格的相邻单元格（上、下、左、右）。

此外，每当他移动到一个开关单元时，所有打开的门单元都会变成关闭的门单元，而所有关闭的门单元都会变成打开的门单元。

确定他是否能从位于起始格的初始状态移动到位于目标格的初始状态，如果可能，找出所需的最少操作次数。

思路

直接BFS然后你会发现做完了，只不过记一下走了几轮而已

E.Reachability Query

题目描述

给你一个无向图，有 NNN 个顶点和 0 条边。
顶点编号为 1,2,…,N1,2,\dots,N1,2,…,N ，最初所有顶点都是白色的。
请处理以下三种类型共 QQQ 个查询：

  • 类型 111 ：添加一条不定向边连接顶点 uuu 和 vvv 。
  • 类型 222 ：如果顶点 vvv 是白色，将其改为黑色；如果是黑色，将其改为白色。
  • 类型 333 ：通过遍历零条或多条边，判断是否可以从顶点 vvv 到达黑色顶点；如果可以，则报告 “是”，否则报告 “否”。

思路

显然DSU可以秒掉连通性的问题，但是题目问的是能不能到达黑色节点，那直接对于每个点记录它在并查集内儿子的黑色节点数量即可。

G.sqrt(n²+n+X)

题目描述

给你一个整数 XXX 。

请找出所有令 n2+n+X\displaystyle \sqrt{n^2+n+X}n2+n+X​ 为整数的整数 nnn 。

思路

简单数学题。

整理：

n2+n+X=m2n^2 + n + X = m^2 n2+n+X=m2

n2+n+(X−m2)=0n^2 + n + (X - m^2) = 0 n2+n+(X−m2)=0

求根公式：

n=−1±1−4(X−m2)2=−1±1−4X+4m22n = \frac{-1 \pm \sqrt{1-4(X-m^2)}}{2} = \frac{-1 \pm \sqrt{1-4X+4m^2}}{2} n=2−1±1−4(X−m2)​​=2−1±1−4X+4m2​​

所以Δ\DeltaΔ必须是一个完平：

1−4X+4m2=k21 - 4X + 4m^2 = k^2 1−4X+4m2=k2

重新整理得到： $$4m2-k2 = 4X-1$$ $$k2-4m2 = 1-4X$$

这可以因式分解为： $$(k-2m)(k+2m) = 1-4X$$

直接枚举1−4X1 - 4X1−4X的因子然后解一个方程组就好了，最后要去重+验证。
